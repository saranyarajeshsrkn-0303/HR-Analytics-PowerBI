📊 HR Analytics Dashboard – Power BI
🔎 Project Overview

This interactive HR Analytics dashboard provides actionable insights into employee attrition, workforce demographics, job satisfaction, and education background trends.

The goal is to help HR teams make data-driven workforce decisions.
